Fri Jun 5, 2020

安装python3

首先下载python3安装包，在csdn上下载可能更快一点。

如果是在服务器上安python3，那可能需要修改镜像源。

app依赖

由于代码里需要import很多依赖，如果直接跑python app.py就会报各种依赖不存在的错误，所以需要先安装所有的依赖：pip install -r requirements.txt
