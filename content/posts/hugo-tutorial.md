```shell
hugo new theme zik-theme
```

可以创建一个新的主题。自己的主题就是从这里开始。来看看初始的theme目录长什么样吧。

```shell
├── LICENSE
├── archetypes
│   └── default.md
├── layouts
│   ├── 404.html
│   ├── _default
│   │   ├── baseof.html
│   │   ├── list.html
│   │   └── single.html
│   ├── index.html
│   └── partials
│       ├── footer.html
│       ├── head.html
│       └── header.html
├── static
│   ├── css
│   └── js
└── theme.toml
```
